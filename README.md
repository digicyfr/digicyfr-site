# Digicyfr Website

A modern, responsive website for Digicyfr built with Next.js, TypeScript, and Tailwind CSS.

## Getting Started

### Development
```bash
# Start development server
npm run dev
# or use the script
./start-dev.sh
```

### Build for Production
```bash
# Build the project
npm run build
# or use the script
./build-site.sh
```
